# Breakout: 2D Arcade Game

An interactive, retro-style 2D Breakout game built from scratch in C++ using an OpenGL-based framework.

## 🎮 Features
* **Real-Time Physics:** Dynamic collision detection and resolution (ball-to-paddle, ball-to-brick) with angle reflections based on hit position.
* **Procedural Layouts:** Bricks are generated dynamically using matrix transformations (translation, scaling) and have different durability levels (colors).
* **Power-ups & Mechanics:** Includes features like multiple ball lives, speed variations, and live score tracking.
* **Smooth Rendering:** Optimized core game loop maintaining a steady frame rate.

## 🛠️ Technologies & Concepts Used
* **Language:** C++
* **Graphics API:** OpenGL (via course framework)
* **Concepts:** Matrix transformations, basic coordinate mapping, view/projection spaces, real-time input handling, and game loop architecture.

## 🚀 How to Run
Open the project in Visual Studio, build it, and run. Use the editor to create a ship, then start the game.
